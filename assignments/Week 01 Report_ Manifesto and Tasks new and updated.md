# Summary:
When I use AI, I intend to do so without taking credit for work I have not accomplished, and without building a total reliance on these tools.
# Plan for Use:
* I plan to use AI in my personal life, academically, and professionally, I plan to use AI. Not doing so would be foolish, as one shouldn't choose to not use a tool simply based on it's origins.
* I will also use it as an assistant, by using it however I please, without letting it govern me or my actions without question
* I intend to use AI frequently, so that I can keep up with how it changes and evolves through time
* I also intend to use AI frugally. Given how many AI tools can be used for free if you put in enough time to learn how to use them, there should be no need to rely on paid, proprietary AI models. Especially when these models can be taken away at any moment, potentially destroying my workflow.
* I intend to use AI in a way that is most customized and easiest for me. It can be pretty easy to use an AI in the "manufacturer reccomended way". However, whenever I use a model, I should spend the time and effort to customize it and make it work in a way that is best for me.

# Limitations and Boundries:
As I use AI, I plan to never cross these boundaries, by endeavoring to use AI:
* Without taking credit for work I haven’t done
* Without creating a total reliance on it (I should be able to perform any task I ask AI to do for me)
* Without allowing it to maintain my social and professional relationships for me
* Without using it to give me information, without my learning it

# Growth Plan
As I continue to use these AI tools into the future, I plan on growing my skillsets and AI fleuncy. For me to accomplish this, going forward I will endeavor to:
* Better understand what AI companies harvest from me when I use their tools. I will accomplish this by reading through AI companies terms of service as they update throughout the years.
* Better understand how to use each AI tool available to me. Which I will accompish by using them. Every year, I will endeavor to use at least one new model, whether that's an LLM, an image generation model, or voice, whatever.
* Better understand how each AI tool works while I use it. I will accomplish this by putting every new model that I use though a comprehensive stress test comprised of a gamut of TMF files designed to test a new model's capabilities, and push it until it breaks. The model will be considered sufficiently tested once it produces two wrong answers in a row.
